# presidenter-policies

Privacy policy and support page for the Presidenter iOS game,
hosted via GitHub Pages.

**Live URLs (after pushing + enabling Pages):**

- https://gokfur.github.io/presidenter-policies/
- https://gokfur.github.io/presidenter-policies/privacy
- https://gokfur.github.io/presidenter-policies/support

These URLs are pasted into App Store Connect as the app's
required Privacy Policy URL and Support URL.

## How to update

Edit any `.md` file. Push to `main`. GitHub Pages auto-rebuilds
within ~1 minute and the live URLs update.

## Files

| File | Purpose |
|---|---|
| `index.md` | Landing page at the root URL. Links to privacy + support. |
| `privacy.md` | Full privacy policy. URL: `/privacy`. |
| `support.md` | Support FAQ + contact info. URL: `/support`. |
| `_config.yml` | Jekyll config (uses the cayman theme). |
| `README.md` | This file. Not rendered as a page. |

## Theme

Uses Jekyll's `cayman` theme — clean, professional, ships with
GitHub Pages, no extra setup. To swap themes, edit `_config.yml`.
